# Guia rápido — iPhone

## Instalar

1. Abra o endereço do GitHub Pages no Safari.
2. Toque em Compartilhar.
3. Escolha **Adicionar à Tela de Início**.
4. Ative **Abrir como App da Web**.
5. Toque em Adicionar.

## Primeira configuração

1. Crie a senha local.
2. Confirme o semestre ativo.
3. Revise as 11 disciplinas.
4. Cadastre os professores.
5. Conecte o Google Calendar em **Mais → Configurações**.
6. Faça um backup JSON após terminar a configuração.

## Cálculo de notas

- **Notas → 1. Avaliações:** crie as atividades.
- **Notas → 2. Cálculo:** monte a fórmula.
- **Notas → 3. Simular:** teste notas futuras.

## Presença

Abra **Mais → Frequência**. O controle do Fatec Flow é pessoal e não substitui o registro oficial do professor ou do SIGA.
